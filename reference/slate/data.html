Data

Data

import { Data } from 'slate'

Data 只是 Immutable.Map 的薄封装，以避免直接依赖 Immutable，并保证向后兼容性。

一个 data 对象可拥有任意的属性。

Data.fromJSON(object: Object) => Data

由 JSON object 构造出一个 data 对象。
